3.19 omp_get_schedule – Obtain the runtime scheduling method

Description:

Obtain the runtime scheduling method. The kind argument will be set to the value omp_sched_static, omp_sched_dynamic, omp_sched_guided or omp_sched_auto. The second argument, chunk_size, is set to the chunk size.

C/C++
Prototype:void omp_get_schedule(omp_sched_t *kind, int *chunk_size);
Fortran:
Interface:subroutine omp_get_schedule(kind, chunk_size)
integer(kind=omp_sched_kind) kind
integer chunk_size
See also:

omp_set_schedule, OMP_SCHEDULE

Reference:

OpenMP specification v4.5, Section 3.2.13.
