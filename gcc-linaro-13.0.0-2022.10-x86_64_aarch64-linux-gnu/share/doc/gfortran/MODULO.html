8.201 MODULO — Modulo function

Description:

MODULO(A,P) computes the A modulo P.

Standard:

Fortran 95 and later

Class:

Elemental function

Syntax:

RESULT = MODULO(A, P)

Arguments:
AShall be a scalar of type INTEGER or REAL.
PShall be a scalar of the same type and kind as A. It shall not be zero. (As a GNU extension, arguments of different kinds are permitted.)
Return value:

The type and kind of the result are those of the arguments. (As a GNU extension, kind is the largest kind of the actual arguments.)

If A and P are of type INTEGER:

MODULO(A,P) has the value R such that A=Q*P+R, where Q is an integer and R is between 0 (inclusive) and P (exclusive).

If A and P are of type REAL:

MODULO(A,P) has the value of A - FLOOR (A / P) * P.

The returned value has the same sign as P and a magnitude less than the magnitude of P.

Example:
program test_modulo
  print *, modulo(17,3)
  print *, modulo(17.5,5.5)

  print *, modulo(-17,3)
  print *, modulo(-17.5,5.5)

  print *, modulo(17,-3)
  print *, modulo(17.5,-5.5)
end program
